Vision-based Page Rank Estimation - Dataset

Context

This dataset was created and used during the student research project "Vision-based Page Rank Estimation" by Timo Denk (mail@timodenk.com) and Samed GÃ¼ner (doktorgibsonyt@gmail.com). It was used to analyze, train Deep Graph Networks (Citations et al. (2016), Citations et al. (2016) and Citations et al. (2016)) and to validate the correlation between page rank and perceived appearance of a website.

Content

Dataset Version 1

The dataset contain 83,296 samples. Each sample represents a mobile and desktop screenshot of a given website as a visitor would perceive within a common browser.

Dataset Version 2

The dataset contains 87,629 samples in total. Each sample has a rank and represents a enriched directed graph of the inner structure of a given website. While nodes are webpages, edges represent hyperlinks between webpages. In addition, each webpage found on a given website has a mobile and desktop version screenshot as a visitor would perceive within a common browser.

Each sample has at most eight nodes. A sample consists of a directed graph in JSON format, whereas nodes contain following information:

  • url
  • given node is the entry point to the website (such as the landing page)
  • client error status (e.g unverified SSL connection)
  • server error code (e.g 404 Not Found)
  • loading time
  • title
  • size
  • list of hyperlinks to other webpages with title
